## 關於專案

這個專案是一個即時天氣應用程式。它提供即時天氣資料，並以簡潔的介面呈現。

專案使用來自中央氣象局的 Open Weather Data 作為資料來源。

### 開發技術

這個應用程式使用以下主要框架與函式庫開發：

* React
* Emotion (styled-components)
* Vite
